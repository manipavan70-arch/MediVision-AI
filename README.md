# MediVision AI

## Project Overview

MediVision AI is an AI-powered X-ray analysis system that detects fractures from uploaded X-ray images, explains the findings, highlights the affected region using Explainable AI, visualizes the injury with a 3D anatomical model, recommends nearby hospitals, and generates downloadable reports.

## Technologies

- Python
- FastAPI
- React
- PyTorch
- OpenCV
- MongoDB
- Three.js

## Features

- X-ray Upload
- AI Fracture Detection
- Heatmap Visualization
- Interactive 3D Anatomical Model
- Hospital Recommendation
- PDF Report
